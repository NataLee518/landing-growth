# Landing Growth

Проект: лендинг с формой заявки и интеграцией с Telegram Bot API.

## Как использовать

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Создайте `.env` с BOT_TOKEN и CHAT_ID
4. Запустите сервер: `npm start`
5. Откройте в браузере: `http://localhost:3000`

## Функции

- Адаптивная верстка
- Hero-блок с CTA
- Форма заявки
- Telegram уведомления
- Простейшие анимации и hover эффекты
